Book 3 Entity Numbers

Note that several entities are left over from Book II (Dwarven Spirit, Brimstone Spider, Lycanthrope, etc.) and some have never been used (Liche). These may not work correctly.

Also note that Lava Slag and Mutated Assassin Stalk share the same entity number. The game evidently uses the first entry with a given number, so if you put entity 99 on a map, it'll be a stalk, not a slag.
